### Что нужно дослать для полного ТЗ в Cursor:

Чтобы Cursor мог написать код для клона складского учета, закупок и цифрового меню, вам необходимо развернуть соответствующие пункты в левом меню и сделать скрины следующих операционных процессов:

**1. Для модуля «Закупка» (Procurement):**
*   Экран создания Заявки на закупку (Purchase Request).
*   Экран конвертации заявки в Заказ поставщику (Purchase Order).
*   Экран цепочки согласования (Approval workflow).

**2. Для модуля «Запасы И Инвентарь» (Склад):**
*   **Приход:** Документ поступления товаров (Приходная накладная / Goods Receipt).
*   **Расход / Перемещение:** Экран списания порчи или перемещения между складами (например, с Главного склада на склад Бара).
*   **Инвентаризация:** Экран ввода фактических остатков при переучете.

**3. Для модуля «Цифровое меню» (DMENU) / Smart Menu:**
*   Конструктор меню (как блюда распределяются по категориям для сканирования по QR-коду).
*   Настройки визуального оформления электронного меню.

**4. Для Интеграции замков:**
*   Если есть отдельный экран маппинга номеров (Room 101 -> Lock ID A-123), он также потребуется. В текущих скринах есть только базовый тумблер включения функции.
