# 0. Видение и границы продукта

## Продукт

**Nafta PMS** — облачная система управления санаторно-гостиничным объектом: бронирование, размещение, счёт гостя, каналы продаж, сервис и медицинский контур санатория.

**Не является:** полноценной бухгалтерией, складским ERP, payroll, налоговым порталом Азербайджана.

## Принципы

1. **Операции отеля — внутри PMS.** Всё, что ресепшен, горничные и менеджмент делают ежедневно, живёт в одной системе.
2. **Финансовый учёт — во внешнем ERP Nafta.** PMS фиксирует *операционные* движения и отдаёт их наружу; не ведёт главную книгу и NAS-план счетов.
3. **Санаторий — first-class.** Мед. записи, лаборатории, процедуры — часть доменной модели, не «надстройка CRM».
4. **AZ compliance — на стыке.** e-qaimə, KKM, реестр туризма — на границе PMS ↔ ERP ↔ гос.сервисы, не копия турецкого e-Fatura/KBS.
5. **Паритет Elektraweb — по операциям, не по ERP.** Берём лучшее из UX отельных модулей; не клонируем турецкую muhasebe.

## Фаза 1 (старт разработки)

**В scope:**

- Справочники отеля (номера, тарифы, питание, отделы, коды дохода)
- Бронирование и управление загрузкой
- Заезд / выезд / проживание
- Фолио гостя: начисления, оплаты, маршрутизация счетов
- Кассовый контур (смены, наличные/карта, предоплаты)
- Channel manager (квоты, цены OTA) — функционально
- Housekeeping
- CRM: гости, задачи, анкеты
- Санаторий: мед. список, лаборатории (базово)
- Ночной аудит (закрытие операционного дня)
- Операционные отчёты для менеджмента

**Вне scope фазы 1** (детально — [14-phase2-roadmap.md](14-phase2-roadmap.md)):

- Бухгалтерские проводки, главная книга, баланс, P&L
- Закупки, полный склад, производственные рецептуры → [16-stock-procurement.md](16-stock-procurement.md)
- POS-зал, SPA-касса → [15-pos-fb-spa.md](15-pos-fb-spa.md)
- AZ e-qaimə, KKM, tourism registry → [17-az-compliance.md](17-az-compliance.md)
- ERP API, статусы fiscal → [18-erp-integration.md](18-erp-integration.md)
- Digital menu, замки — пакет P2-E в roadmap
- Турецкие модули: KBS, e-Fatura, 5651

## Пользователи (роли)

| Роль | Основная работа в системе |
|------|---------------------------|
| Ресепшен | Брони, check-in/out, фолио, касса |
| Ночной аудитор | Закрытие дня, контроль ошибок |
| Менеджер по продажам / revenue | Квоты, тарифы, OTA, отчёты |
| Горничная / супервайзер HK | Статусы номеров, задания |
| Врач / мед. регистратура | Мед. записи, назначения анализов |
| CRM / качество | Жалобы, анкеты, задачи |
| Администратор отеля | Справочники, пользователи, настройки |
| Бухгалтерия | **Внешнее ERP**; в PMS — просмотр выгрузок/статусов (фаза 2) |

## Отличие от лицензии Elektraweb у Nafta

В лицензии включены ACC, STOCK, POS, SPA, Procurement и др. По фактическому использованию (манифест 292 экрана) Nafta **активно пользуется** PMS, кассой, каналами, CRM/мед., HK. Модули **ACC-GL, закупки, POS-зал, SPA-касса** почти не раскрыты в интерфейсе — для нашего продукта это подтверждение стратегии «операции в PMS, финансы в ERP».
