# 8. Передача данных во внешнее ERP (концептуально)

> Без форматов API и полей БД — только **какие бизнес-события** PMS отдаёт ERP Nafta.

## Принцип

PMS — **источник правды** по гостиничным операциям.  
ERP — **источник правды** по учёту, налогам, платежам поставщикам, закрытию периода.

Связь **односторонняя сначала**: PMS → ERP. Обратно из ERP в PMS (например, статус «счёт проведён») — фаза 2.

## События и пакеты

### E1. Закрытие операционного дня (после night audit)

**Когда:** раз в сутки, после успешного night audit.

**Содержание (агрегаты):**

| Блок | Пример строк |
|------|----------------|
| Выручка | Код ROOM, отдел Accommodation, 12 500 AZN |
| Выручка | Код MEDICAL, отдел Medical, 3 200 AZN |
| Платежи | Наличные 5 000, карта 8 000, банк 2 000 |
| НДС-метки | Если заданы на коде дохода — для ERP |

### E2. Выписанный счёт (invoice)

**Когда:** счёт гостю/компании переведён в статус «выставлен».

**Содержание:**

- Номер, дата, контрагент (гость / компания / агентство).  
- Строки: описание, сумма, НДС, код дохода.  
- Ссылка на бронь / проживание.

### E3. Платёж

**Когда:** принят платёж на фолио или по city ledger.

**Содержание:**

- Сумма, валюта, способ, дата.  
- Привязка к фолио или агентству.  
- Касса / смена (для сверки наличных).

### E4. City ledger (периодически)

**Когда:** по запросу или вместе с E1.

**Содержание:**

- Остаток по агентству на дату.  
- Оборот за период: начислено / оплачено.

### E5. Справочники (редко, по изменению)

**Когда:** изменились коды дохода, отделы, ставки НДС для маппинга.

**Содержание:**

- Синхронизация кодов PMS ↔ кодов ERP.

## Что ERP делает сама (не в PMS)

- Двойная запись, проводки дебет/кредит.  
- Закрытие месяца, inventory в бух. смысле.  
- Зарплата, ОС, закупки.  
- e-qaimə submission (может получать из E2).  
- Сверка с банком.

## Что не копируем из Elektraweb ACC

| Elektraweb | Наш подход |
|------------|------------|
| Экспорт Sage | Аналог: **ваш ERP API** |
| Bavel / TR e-invoice | AZ e-qaimə через ERP или fiscal gateway |
| Muhasebe fişleri | **Нет в PMS** |
| Импорт Excel в GL | **Нет** — импорт справочников услуг опционально в PMS |
