# 03. Зал, заказ, KDS

## 3.1 Карта зала (Floor plan)

### Экраны

| Экран | Роль | Содержание |
|-------|------|------------|
| Floor map | Официант | Сетка/план столов, цвет статуса |
| Table detail | Официант | Открытый чек, время, сумма |
| Walk-in | Официант | Чек без стола (барная стойка) |

### Действия

| Действие | Правило |
|----------|---------|
| Open ticket | Стол FREE или RESERVED (с бронью) |
| Transfer table | Перенос открытого чека |
| Merge tables | Объединение чеков (manager) |
| Split ticket | Разделение на 2 чека (manager) |
| View reservation | Ссылка на бронь из календаря |

### Цвета статусов

| Статус | UI |
|--------|-----|
| FREE | Зелёный |
| OCCUPIED | Занят (сумма) |
| RESERVED | Синий (время брони) |
| DIRTY | Серый (не сажать) |

## 3.2 Оформление заказа (Waiter POS)

### UX-поток

1. Выбор outlet (если несколько).
2. Стол или walk-in.
3. Каталог: категории слева, плитки блюд, поиск, штрихкод (фаза 2).
4. Модификаторы в модалке.
5. Комментарий кухне на строку.
6. **Hold** — не отправлять на KDS.
7. **Fire** — отправить новые строки на KDS.
8. Пречек — печать/PDF без закрытия.

### Бизнес-правила

| Правило | Описание |
|---------|----------|
| Void line | Только manager; причина обязательна |
| Discount ticket | % или сумма; manager; причина |
| Min order | Опционально на outlet |
| Service charge % | Авто на чек (настройка outlet) |

### Связь in-house

Панель «На номер»: поиск по комнате/фамилии → список из PMS API → привязка `reservationId` к ticket (не оплата до close).

### Станции

| Station | Примеры |

Маршрутизация: `MenuItem.station` → очередь.

|------|--|

| Действие | Эффект |

## 3.4 Календарь бронирований (в fb-pos)

**В hotel-pms** уже есть lite `/pos/calendar`.

**В fb-pos (полный):**

| Функция | Описание |
|---------|----------|
| Бронь стола на время | party size, guest phone |
| Link PMS reservation | Автоподстановка имени |
| Check-in брони | Открыть ticket с RESERVED → OCCUPIED |
| No-show брони | Освободить слот |

Синхронизация PMS ↔ fb-pos для слотов — **фаза 2** (один source of truth: fb-pos для ресторана, PMS read-only или webhook).
